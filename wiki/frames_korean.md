# JavaScript 프레임 (Frames) - 웹 페이지 내에서의 다중 콘텐츠 표시

## 개요
JavaScript에서 "프레임"은 웹 페이지 내에 여러 개의 독립적인 콘텐츠 영역을 생성하는 방법입니다. 프레임을 사용하면 하나의 웹 페이지 안에서 여러 문서나 웹 페이지를 동시에 표시할 수 있습니다. 이 기능은 특히 복잡한 사용자 인터페이스를 구축할 때 유용합니다.

## 문서화

### 목적
프레임은 여러 웹 페이지를 동시에 표시할 수 있게 하여 사용자에게 더 나은 탐색 경험을 제공합니다. 프레임을 사용하면 페이지의 일부를 업데이트 하거나 새로 고침 없이도 콘텐츠를 변경할 수 있습니다.

### 사용법
JavaScript에서 프레임을 사용하기 위해 `<iframe>` 태그를 주로 활용합니다. 이 태그를 사용하여 다른 페이지를 현재 페이지의 일부로 포함할 수 있습니다. 

### 세부정보
- **기본 구조**: `<iframe src="URL" width="넓이" height="높이"></iframe>`
- **src**: 표시할 페이지의 URL을 지정합니다.
- **width/height**: 프레임의 크기를 설정합니다.

프레임은 서로 다른 출처의 콘텐츠를 포함할 수 있으나, 보안 상의 이유로 Cross-Origin Resource Sharing (CORS) 정책에 따라 제한이 있을 수 있습니다.

## 예제

### 기본 사용 예제
```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>프레임 예제</title>
</head>
<body>
    <h1>내 웹사이트</h1>
    <iframe src="https://www.example.com" width="600" height="400" title="Example Frame"></iframe>
</body>
</html>
```

이 예제는 `https://www.example.com` 웹사이트를 현재 페이지 내 600x400 픽셀 크기의 프레임에 표시합니다.

## 설명

### 일반적인 오류 및 주의사항
- **보안 문제**: 다른 출처의 콘텐츠를 포함할 때는 CORS 정책을 존중해야 하며, 일부 웹사이트는 iframe 내에서 로드되는 것을 차단할 수 있습니다.
- **SEO 문제**: 프레임 내의 콘텐츠는 검색 엔진이 인덱스하기 어려울 수 있으며, 검색 엔진 최적화(SEO)에 부정적인 영향을 미칠 수 있습니다.
- **반응형 디자인**: 프레임은 모바일 장치에서 비율 유지에 어려움을 겪을 수 있으므로, CSS를 사용하여 모바일 친화적인 디자인을 적용하는 것이 좋습니다.

## 한 줄 요약
JavaScript에서 프레임은 웹 페이지 내에 다중 콘텐츠를 표시하는 데 사용되는 `<iframe>` 태그를 통해 구현됩니다.
