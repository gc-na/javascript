# Uint8Array في جافا سكريبت: كل ما تحتاج لمعرفته

## ملخص
`Uint8Array` هو نوع من مصفوفات النوع الثابت في جافا سكريبت، يُستخدم لتخزين الأعداد الصحيحة غير السالبة بصيغة 8 بت. يوفر هذا النوع من المصفوفات أداءً عاليًا عند التعامل مع البيانات الثنائية مثل الصور والصوت.

## الوثائق
`Uint8Array` هو جزء من واجهة `TypedArray` في جافا سكريبت. يتم استخدامه لإنشاء مصفوفات من الأعداد الصحيحة غير السالبة، حيث يمكن أن تمثل كل قيمة 0 إلى 255. يتميز `Uint8Array` بقدرته على توفير أداء متفوق في معالجة البيانات الكبيرة، مما يجعله خيارًا شائعًا في تطبيقات معالجة البيانات.

### الاستخدام
يمكنك إنشاء `Uint8Array` بعدة طرق:
1. **إنشاء مصفوفة جديدة:** 
   ```javascript
   const arr = new Uint8Array(10); // مصفوفة تحتوي على 10 عناصر، جميعها 0
   ```
2. **إنشاء مصفوفة من مصفوفة أخرى:**
   ```javascript
   const arr = new Uint8Array([1, 2, 3, 256]); // القيم فوق 255 سيتم تقليصها إلى 0
   ```
3. **إنشاء مصفوفة من Buffer:**
   ```javascript
   const buffer = new ArrayBuffer(8);
   const arr = new Uint8Array(buffer); // مصفوفة مرتبطة بـ Buffer
   ```

### التفاصيل
- **الطول:** يمكنك الحصول على طول مصفوفة `Uint8Array` باستخدام الخاصية `length`.
- **الوصول إلى القيم:** يمكنك الوصول إلى القيم باستخدام الفهارس، مثل `arr[0]`.
- **طرق مفيدة:** تشمل `set()` لإعداد قيم جديدة، و `subarray()` لإنشاء جزء من المصفوفة.
  
## أمثلة
### مثال 1: إنشاء مصفوفة وإضافة القيم
```javascript
const arr = new Uint8Array(5);
arr[0] = 10;
arr[1] = 20;
console.log(arr); // Uint8Array(5) [10, 20, 0, 0, 0]
```

### مثال 2: استخدام `set()`
```javascript
const arr = new Uint8Array(5);
arr.set([5, 15]); 
console.log(arr); // Uint8Array(5) [5, 15, 0, 0, 0]
```

### مثال 3: استخدام `subarray()`
```javascript
const arr = new Uint8Array([1, 2, 3, 4, 5]);
const subArr = arr.subarray(1, 4);
console.log(subArr); // Uint8Array(3) [2, 3, 4]
```

## الشرح
عند العمل مع `Uint8Array`، هناك بعض النقاط التي يجب مراعاتها:
- القيم التي تتجاوز 255 يتم تقليصها إلى 0. على سبيل المثال، `256` سيتم تمثيله كـ `0`.
- `Uint8Array` لا يقبل القيم السالبة، لذا أي محاولة لإضافة قيمة سالبة ستؤدي إلى تحويلها إلى `0`.
- قد يكون هناك تحذير بشأن الأداء عند استخدام `Uint8Array` مع بيانات ليست ثنائية، لذا يجب استخدامه بحذر.

## ملخص
`Uint8Array` هو نوع من مصفوفات جافا سكريبت يوفر أداءً عاليًا لتخزين الأعداد الصحيحة غير السالبة.
