# حدث USBConnectionEvent في JavaScript: كل ما تحتاج معرفته

## ملخص
يعد حدث USBConnectionEvent من الأحداث المهمة في JavaScript التي تتعامل مع اتصالات الأجهزة عبر واجهة USB. يوفر هذا الحدث معلومات عن حالة اتصال الأجهزة USB، مما يساهم في تحسين تجربة المستخدم عند التعامل مع الأجهزة المتصلة.

## الوثائق
### الغرض
حدث USBConnectionEvent يُستخدم للإشارة إلى تغييرات في حالة اتصال الأجهزة عبر USB. يمكن أن يحدث هذا الحدث عند توصيل أو فصل جهاز USB عن النظام، مما يسمح للمطورين بالتفاعل مع هذه التغييرات بشكل ديناميكي.

### الاستخدام
يمكن الاستماع لحدث USBConnectionEvent باستخدام `addEventListener`، مما يمكّن المطورين من تنفيذ تعليمات برمجية معينة عند حدوث تغييرات في حالة الاتصال. يتضمن الحدث معلومات حول الجهاز المتصل أو المفصول.

#### بناء الجملة:
```javascript
navigator.usb.addEventListener('connect', (event) => {
    // الكود الذي يتم تنفيذه عند توصيل جهاز USB
});

navigator.usb.addEventListener('disconnect', (event) => {
    // الكود الذي يتم تنفيذه عند فصل جهاز USB
});
```

### التفاصيل
- **الخصائص**: يحتوي حدث USBConnectionEvent على خصائص مثل `device`، والتي تمثل الجهاز المتصل.
- **الدعم**: تأكد من أن واجهة USB مدعومة في المتصفح المستخدم، حيث قد يختلف الدعم بين المتصفحات المختلفة.

## الأمثلة
### مثال 1: الاستماع لتوصيل جهاز USB
```javascript
navigator.usb.addEventListener('connect', (event) => {
    console.log('جهاز USB متصل:', event.device);
});
```

### مثال 2: الاستماع لفصل جهاز USB
```javascript
navigator.usb.addEventListener('disconnect', (event) => {
    console.log('جهاز USB مفصول:', event.device);
});
```

## الشرح
### الأخطاء الشائعة
- **عدم دعم المتصفح**: تأكد من أن المتصفح يدعم واجهة USB، حيث لا تدعم بعض المتصفحات هذه الميزة.
- **إعدادات الأمان**: قد تحتاج إلى التعامل مع إعدادات الأمان الخاصة بالمتصفح، حيث قد يتم حظر الوصول إلى واجهة USB في بعض الحالات.

### ملاحظات إضافية
- يُفضل استخدام `navigator.usb` في بيئات موثوقة، مثل تطبيقات الويب التي تتطلب إذنًا خاصًا للوصول إلى الأجهزة.
- تأكد من معالجة الأخطاء بشكل جيد عند التعامل مع الأجهزة، مثل الأخطاء الناتجة عن عدم العثور على جهاز USB.

## ملخص جملة واحدة
حدث USBConnectionEvent في JavaScript يُتيح للمطورين الاستجابة لتغييرات حالة اتصال الأجهزة عبر USB بشكل ديناميكي وفعال.
