# حدث WheelEvent في جافا سكريبت: دليل شامل

## الملخص
حدث `WheelEvent` في جافا سكريبت هو أحد الأحداث التي تُستخدم للتعامل مع حركات العجلة (الماوس أو لوحة اللمس)، مما يسمح للمطورين بالتفاعل مع المستخدم بطريقة أكثر ديناميكية عند التمرير أو تغيير مستوى التكبير.

## الوثائق
### الغرض
`WheelEvent` هو نوع من الأحداث يتم إنشاؤه عندما يقوم المستخدم بالتمرير باستخدام عجلة الماوس أو من خلال استخدام إيماءات التمرير على الأجهزة التي تدعم اللمس. يهدف هذا الحدث إلى تقديم معلومات دقيقة حول حركة العجلة، مما يمكّن المطورين من تنفيذ تفاعلات خاصة استجابةً لهذه الحركات.

### الاستخدام
يمكن استخدام `WheelEvent` في عناصر HTML المختلفة، ويمكن التفاعل معه من خلال إضافة مستمعين للأحداث. يتم استدعاء حدث `wheel` في حال تم التمرير.

### تفاصيل
- **الأحداث المرتبطة**: `wheel`, `scroll`
- **الخصائص الرئيسية**:
  - `deltaX`: مقدار الحركة الأفقية.
  - `deltaY`: مقدار الحركة العمودية.
  - `deltaZ`: مقدار الحركة في الاتجاه العمودي الثالث (نادراً ما يُستخدم).
  - `deltaMode`: وضع الحركة (على سبيل المثال، بكسل، سطر، أو صفحة).

### كيفية الاستخدام
لإضافة مستمع حدث `WheelEvent` إلى عنصر، يمكنك استخدام الكود التالي:

```javascript
const element = document.getElementById('myElement');
element.addEventListener('wheel', function(event) {
    console.log('تم التمرير:', event.deltaY);
});
```

## أمثلة
### مثال 1: تسجيل حركة التمرير
```javascript
const myDiv = document.getElementById('scrollableDiv');
myDiv.addEventListener('wheel', (event) => {
    event.preventDefault(); // لمنع السلوك الافتراضي
    console.log(`حركة التمرير: ${event.deltaY}`);
});
```

### مثال 2: تغيير مستوى التكبير
```javascript
const zoomableImage = document.getElementById('zoomableImage');
let scale = 1;

zoomableImage.addEventListener('wheel', (event) => {
    event.preventDefault();
    scale += event.deltaY * -0.01; // تعديل مستوى التكبير بناءً على حركة العجلة
    scale = Math.min(Math.max(.125, scale), 4); // تحديد حدود التكبير
    zoomableImage.style.transform = `scale(${scale})`;
});
```

## الشرح
### الأخطاء الشائعة
- **عدم استخدام `event.preventDefault()`**: عند التعامل مع أحداث التمرير، قد تحتاج إلى منع السلوك الافتراضي (مثل تمرير الصفحة) إذا كنت ترغب في تنفيذ سلوك مخصص.
- **تجاهل خصائص `deltaMode`**: قد يؤدي الفشل في فهم كيفية عمل `deltaMode` إلى نتائج غير متوقعة. يجب أن تكون على دراية بأن القيم المختلفة تمثل وحدات قياس مختلفة.

### ملاحظات إضافية
- تأكد من اختبار سلوك `WheelEvent` عبر متصفحات مختلفة، حيث يمكن أن تختلف التجربة بين المتصفحات.
- يمكن أن تؤثر الإضافات (مثل مكتبات JavaScript) على كيفية عمل `WheelEvent`، لذا تأكد من التحقق من التوافق.

## ملخص من سطر واحد
يعتبر حدث `WheelEvent` في جافا سكريبت أداة قوية للتحكم في تفاعلات التمرير، مما يمكّن المطورين من تحسين تجربة المستخدم بشكل ديناميكي.
