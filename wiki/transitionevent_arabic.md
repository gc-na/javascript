# حدث الانتقال (TransitionEvent) في جافا سكريبت

## ملخص
حدث الانتقال (TransitionEvent) هو حدث يُستخدم في جافا سكريبت للكشف عن بداية ونهاية الانتقالات (transitions) في العناصر التي تستخدم CSS. يُتيح هذا الحدث للمطورين تنفيذ أكواد معينة عند حدوث هذه الفعاليات، مما يعزز من تجربة المستخدم.

## الوثائق
### الغرض
يُعتبر حدث الانتقال جزءًا من واجهة برمجة التطبيقات (API) الخاصة بالأحداث في جافا سكريبت، ويُستخدم لتتبع حالات الانتقال في العناصر عند تطبيق تأثيرات CSS مثل التلاشي أو التمدد.

### الاستخدام
يمكن الاستماع إلى حدث الانتقال عن طريق إضافة مُعالج حدث إلى العنصر المستهدف. عند بدء أو انتهاء الانتقال، يتم إطلاق هذا الحدث.

### التفاصيل
- **الخصائص**:
  - `propertyName`: اسم خاصية CSS التي تم تغييرها.
  - `elapsedTime`: الوقت الذي استغرقه الانتقال حتى الآن.
  - `pseudoElement`: يُرجع اسم العنصر الزائف (إذا كان موجودًا).

### الهيكل
```javascript
element.addEventListener('transitionend', function(event) {
    // تنفيذ الكود عند انتهاء الانتقال
});
```

## الأمثلة
### مثال 1: الاستماع لحدث انتهاء الانتقال
```javascript
const box = document.querySelector('.box');

box.addEventListener('transitionend', (event) => {
    console.log(`تم الانتهاء من الانتقال للخاصية: ${event.propertyName}`);
});

// بدء الانتقال
box.style.transition = 'all 2s';
box.style.transform = 'translateX(100px)';
```

### مثال 2: التحقق من وقت الانتقال
```javascript
const button = document.querySelector('.button');

button.addEventListener('transitionend', (event) => {
    console.log(`الوقت المستغرق للانتقال: ${event.elapsedTime} ثانية`);
});

// بدء الانتقال
button.style.transition = 'opacity 1s';
button.style.opacity = '0';
```

## الشرح
### الأخطاء الشائعة
1. **عدم دعم المتصفح**: تأكد من أن المتصفح المستخدم يدعم أحداث الانتقال، حيث قد لا تكون بعض المتصفحات القديمة قادرة على التعامل مع هذه الأحداث بشكل صحيح.
2. **عدم إضافة خاصية الانتقال بشكل صحيح**: تأكد من أنك قمت بتطبيق تأثيرات CSS بشكل صحيح، وإلا فلن يتم إطلاق الحدث.
3. **عدم التعامل مع جميع الخصائص**: إذا كنت قد قمت بتغيير أكثر من خاصية CSS في نفس الوقت، قد تحتاج إلى التحقق من `event.propertyName` لمعرفة أي خاصية انتهى انتقالها.

## ملخص جملة واحدة
يعد حدث الانتقال (TransitionEvent) في جافا سكريبت أداة قوية تسمح للمطورين بالتفاعل مع تأثيرات الانتقال في CSS لتحسين تجربة المستخدم.
