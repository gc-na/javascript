# TimeRanges في JavaScript: دليل شامل للمطورين

## ملخص
تُستخدم كائنات `TimeRanges` في JavaScript لتمثيل مجموعة من الفترات الزمنية، وغالبًا ما تكون مرتبطة بعناصر الوسائط مثل الفيديو. هذه الكائنات مفيدة لتحديد الأجزاء القابلة للتشغيل من الوسائط.

## الوثائق
تُعد `TimeRanges` جزءًا من واجهة `HTMLMediaElement` في JavaScript، وتُستخدم لتحديد الفترات الزمنية التي يمكن تشغيلها من عنصر الوسائط. يتضمن `TimeRanges` مجموعة من الفترات الزمنية، ويمكن الوصول إليها من خلال خصائص مثل `video.played` أو `audio.played`.

### الغرض
يتيح لك `TimeRanges` معرفة الأجزاء من الوسائط التي تم تحميلها أو تشغيلها، مما يساعد في إدارة تحميل الوسائط بشكل أكثر كفاءة.

### الاستخدام
يمكن استخدام `TimeRanges` مع العناصر التي تدعم الوسائط مثل `<video>` و `<audio>`. يمكنك الوصول إلى فترات التشغيل باستخدام الخصائص المتاحة.

### التفاصيل
- **الخصائص**:
  - `length`: عدد الفترات الزمنية في الكائن.
  - `start(index)`: تُرجع بداية الفترة الزمنية في الفهرس المحدد.
  - `end(index)`: تُرجع نهاية الفترة الزمنية في الفهرس المحدد.

- **الأساليب**:
  - لا يحتوي `TimeRanges` على طرق (methods) مباشرة، بل يعتمد على الخصائص فقط.

## أمثلة
### مثال 1: الوصول إلى فترات التشغيل
```javascript
const video = document.querySelector('video');

video.addEventListener('loadedmetadata', () => {
    const ranges = video.played;
    console.log(`عدد الفترات: ${ranges.length}`);

    for (let i = 0; i < ranges.length; i++) {
        console.log(`الفترة ${i + 1}: تبدأ عند ${ranges.start(i)} وتنتهي عند ${ranges.end(i)}`);
    }
});
```

### مثال 2: تحديد الأجزاء القابلة للتشغيل
```javascript
const audio = document.querySelector('audio');

audio.addEventListener('loadeddata', () => {
    const playedRanges = audio.played;
    if (playedRanges.length > 0) {
        console.log(`تم تشغيل الأجزاء من ${playedRanges.start(0)} إلى ${playedRanges.end(0)}`);
    }
});
```

## الشرح
### الأخطاء الشائعة
- **عدم وجود فترات**: قد يحدث أن تكون فترات التشغيل غير متاحة إذا لم يتم تحميل الوسائط بشكل صحيح.
- **التعامل مع الفهارس**: تأكد من أن الفهارس المستخدمة متوافقة مع عدد الفترات المتاحة لتجنب الأخطاء.

### ملاحظات إضافية
- `TimeRanges` مفيدة لتطبيقات الويب التي تحتاج إلى إدارة الوسائط بشكل ديناميكي، خاصة عند التعامل مع ملفات كبيرة أو تدفقات مباشرة.

## ملخص جملة واحدة
تُستخدم كائنات `TimeRanges` في JavaScript لتحديد الفترات الزمنية القابلة للتشغيل في عناصر الوسائط مثل الفيديو والصوت.
