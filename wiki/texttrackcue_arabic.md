# نصوص الترجمة: TextTrackCue في JavaScript

## ملخص
TextTrackCue هو كائن في JavaScript يُستخدم لإدارة الترجمة النصية في الفيديوهات، ويسمح للمطورين بإضافة نصوص توضيحية أو ترجمات إلى الوسائط المتعددة.

## التوثيق
### الغرض
TextTrackCue يُستخدم لتمثيل نصوص الترجمة أو التوضيحات المستخدمة في الفيديوهات، مما يُحسن تجربة المستخدمين الذين يحتاجون إلى النصوص المساعدة لفهم المحتوى.

### الاستخدام
يمكن استخدام TextTrackCue في تطوير تطبيقات الويب التي تتعامل مع الوسائط المتعددة. يتم إنشاؤه عادةً عند التعامل مع كائنات TextTrack، حيث يمثل كل نص ترجمة أو توضيح معين.

### التفاصيل
- **الخصائص**:
  - `startTime`: الوقت الذي يبدأ فيه عرض النص.
  - `endTime`: الوقت الذي ينتهي فيه عرض النص.
  - `text`: النص الفعلي الذي سيتم عرضه.
  
- **الطرق**:
  - `getCueAsHTML()`: تعيد تمثيل HTML للنص، مما يسمح بتنسيق النص بشكل أفضل.

## أمثلة
### مثال بسيط على استخدام TextTrackCue
```javascript
// إنشاء كائن TextTrackCue
const cue = new TextTrackCue(0, 5, "مرحبًا بكم في الفيديو!");

// إضافة الكائن إلى TextTrack
const track = video.textTracks[0];
track.addCue(cue);
```

### مثال مع استخدام getCueAsHTML
```javascript
const cue = new TextTrackCue(0, 10, "شاهد هذا المحتوى الرائع!");
console.log(cue.getCueAsHTML()); // يعرض النص بتنسيق HTML
```

## الشرح
### الأخطاء الشائعة
- **عدم التحقق من وجود نصوص الترجمة**: يجب التأكد من أن كائن TextTrack موجود قبل إضافة TextTrackCue.
- **توقيت غير صحيح**: تأكد من أن `startTime` و `endTime` ضمن نطاق زمن الفيديو وإلا لن يظهر النص.

### ملاحظات إضافية
- TextTrackCue يُعتبر جزءًا من واجهة WebVTT ويعمل بشكل أفضل مع الفيديوهات التي تدعم الترجمة النصية.
- يجب أن تكون النصوص واضحة ومفيدة لتقليل الإرباك لدى المستخدمين.

## ملخص جملة واحدة
TextTrackCue في JavaScript هو كائن ضروري لإدارة النصوص الترجمة في الفيديوهات، مما يُحسن من تجربة المستخدمين.
