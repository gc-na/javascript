# Uint16Array في JavaScript: فهم المصفوفات غير القابلة للتغيير

## الملخص
`Uint16Array` هو نوع خاص من المصفوفات في JavaScript يستخدم لتخزين الأعداد الصحيحة غير السالبة في مدى من 0 إلى 65535. يتيح لك هذا النوع من المصفوفات التعامل مع البيانات الثنائية بكفاءة.

## الوثائق
`Uint16Array` هو جزء من واجهة `TypedArray` في JavaScript، ويتيح لك إنشاء مصفوفة من الأعداد الصحيحة غير السالبة بطول 16 بت. يمكن استخدامه لتخزين البيانات الرقمية بشكل أكثر كفاءة، مما يجعله مفيدًا في التطبيقات التي تتطلب معالجة البيانات الثنائية، مثل الألعاب أو معالجة الصور.

### الاستخدام:
يمكنك إنشاء `Uint16Array` بعدة طرق:
1. **إنشاء مصفوفة فارغة**:
   ```javascript
   const arr = new Uint16Array(5);
   ```

2. **إعادة تهيئة مصفوفة من مصفوفة أخرى**:
   ```javascript
   const arrFromArray = new Uint16Array([1, 2, 3, 4, 5]);
   ```

3. **إنشاء مصفوفة من بيانات `ArrayBuffer`**:
   ```javascript
   const buffer = new ArrayBuffer(8);
   const arrFromBuffer = new Uint16Array(buffer);
   ```

### الخصائص:
- **طول المصفوفة**: يمكنك الوصول إلى طول المصفوفة باستخدام `length`.
- **الوصول إلى العناصر**: يمكنك الوصول إلى العناصر باستخدام الفهارس، مثل `arr[0]`.
- **التحويل إلى مصفوفة عادية**: يمكنك تحويل `Uint16Array` إلى مصفوفة عادية باستخدام `Array.from(arr)`.

## الأمثلة
### مثال 1: إنشاء مصفوفة وتعيين القيم
```javascript
const uint16Array = new Uint16Array(3);
uint16Array[0] = 1000;
uint16Array[1] = 2000;
uint16Array[2] = 3000;
console.log(uint16Array); // Uint16Array(3) [ 1000, 2000, 3000 ]
```

### مثال 2: استخدام `ArrayBuffer`
```javascript
const buffer = new ArrayBuffer(4); // 4 بايت = 2 عددين
const uint16Array = new Uint16Array(buffer);
uint16Array[0] = 65535; // الحد الأقصى
uint16Array[1] = 40000;
console.log(uint16Array); // Uint16Array(2) [ 65535, 40000 ]
```

## الشرح
عند استخدام `Uint16Array`، يجب مراعاة بعض النقاط:
- **الحدود القيمية**: القيم التي يمكن تخزينها تتراوح من 0 إلى 65535. أي قيمة خارج هذا النطاق سيتم تحويلها إلى 0.
- **الذاكرة**: `Uint16Array` يستخدم مساحة ذاكرة ثابتة، لذا تأكد من حجم `ArrayBuffer` الذي تقوم بإنشائه.
- **الأداء**: `TypedArrays` عادة ما تكون أسرع من المصفوفات العادية في العمليات الرياضية، مما يجعلها مثالية للتطبيقات التي تتطلب الأداء العالي.

## ملخص
`Uint16Array` هو نوع من المصفوفات في JavaScript مصمم لتخزين الأعداد الصحيحة غير السالبة بكفاءة، مما يجعله خيارًا ممتازًا لتطبيقات معالجة البيانات الثنائية.
