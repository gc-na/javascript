# JavaScript scrollBy 메서드: 페이지 스크롤을 제어하는 방법

## 개요
`scrollBy`는 JavaScript에서 사용되는 메서드로, 현재 스크롤 위치를 기준으로 지정된 수치만큼 스크롤을 이동시키는 기능을 제공합니다. 이 메서드는 브라우저의 뷰포트를 수평 및 수직 방향으로 움직이는 데 유용하며, 사용자 경험을 향상시키는 데 사용됩니다.

## 문서화

### 목적
`scrollBy` 메서드는 현재 스크롤 위치를 기준으로 페이지 내용을 일정한 양만큼 스크롤하며, 애니메이션 효과와 함께 사용할 수 있습니다. 이를 통해 부드러운 스크롤링 효과를 구현할 수 있습니다.

### 사용법
`scrollBy` 메서드는 `window` 객체에서 호출되며, 두 개의 매개변수를 받습니다:

```javascript
window.scrollBy(x, y);
```

- **x**: 수평 방향으로 이동할 픽셀 수. 양수 값은 오른쪽으로, 음수 값은 왼쪽으로 스크롤합니다.
- **y**: 수직 방향으로 이동할 픽셀 수. 양수 값은 아래쪽으로, 음수 값은 위쪽으로 스크롤합니다.

### 세부사항
- `scrollBy`는 비동기적으로 작동하며, 스크롤을 즉시 반영합니다.
- 브라우저의 뷰포트가 이동하며, 스크롤 위치는 `document.documentElement.scrollTop` 또는 `window.scrollY`로 확인할 수 있습니다.
- 스크롤 애니메이션을 위해 `requestAnimationFrame`과 함께 사용하면 보다 부드러운 효과를 얻을 수 있습니다.

## 예제

### 기본 사용 예제

#### 수직 스크롤
```javascript
// 100픽셀 아래로 스크롤
window.scrollBy(0, 100);
```

#### 수평 스크롤
```javascript
// 50픽셀 오른쪽으로 스크롤
window.scrollBy(50, 0);
```

#### 수평 및 수직 스크롤
```javascript
// 30픽셀 오른쪽, 200픽셀 아래로 스크롤
window.scrollBy(30, 200);
```

## 설명
- **스크롤 범위**: `scrollBy` 메서드는 지정된 값만큼 스크롤하므로, 스크롤 위치가 페이지의 가장자리에 도달할 경우 스크롤이 더 이상 진행되지 않을 수 있습니다.
- **브라우저 호환성**: 대부분의 최신 브라우저에서 지원되지만, 구형 브라우저에서는 기능이 제한적일 수 있습니다.
- **부드러운 스크롤링**: CSS의 `scroll-behavior: smooth;` 속성을 사용하면 스크롤을 부드럽게 만들 수 있지만, 이는 모든 브라우저에서 동일하게 작동하지 않을 수 있습니다.

## 한 줄 요약
`scrollBy` 메서드는 현재 스크롤 위치를 기준으로 지정된 픽셀만큼 페이지를 수평 또는 수직으로 스크롤하는 JavaScript 기능입니다.
